# Lab 02 – Red Corporativa Multisede

## Objetivo
Diseñar e implementar una red corporativa multisede que conecte una Oficina Central (HQ) y un Centro de Distribución (DC).

## Escenario
- Redes LAN segmentadas por área
- Enlace WAN punto a punto entre sedes
- Infraestructura corporativa realista

## Tecnologías aplicadas
- IPv4 con VLSM
- IPv6
- Enrutamiento estático
- Servidor TFTP para backup de configuraciones
- Seguridad básica en routers y switches

## Validación
- Pruebas de conectividad con ping (IPv4 e IPv6)
- Pruebas de ruta con traceroute
- Verificación en modo simulación
