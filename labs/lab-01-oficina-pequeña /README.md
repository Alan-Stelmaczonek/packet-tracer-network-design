# Lab 01 – Red LAN para Oficina Pequeña

## Objetivo
Diseñar e implementar una red LAN funcional para una oficina pequeña, asegurando conectividad entre dispositivos finales y correcta configuración del gateway.

## Topología
- 1 Router
- 1 Switch principal
- 2 Switches de acceso
- Estaciones de trabajo distribuidas por área

## Configuración realizada
- Direccionamiento IPv4 estático
- Configuración de gateway
- IP de gestión en switches
- Banners de advertencia

## Pruebas
- Ping entre estaciones de distintas áreas
- Ping hacia el gateway
- Acceso a switches mediante IP de gestión
