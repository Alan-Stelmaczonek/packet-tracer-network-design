# Packet Tracer – Network Design Projects

Repositorio que contiene proyectos de diseño e implementación de redes desarrollados en Cisco Packet Tracer, basados en escenarios reales de oficina y entornos corporativos.

## Tecnologías
- Cisco Packet Tracer
- Routing & Switching (nivel CCNA)
- IPv4 (VLSM) e IPv6
- Enrutamiento estático
- Seguridad básica de red
- Validación mediante ping y traceroute

## Laboratorios

### Lab 01 – Red LAN para Oficina Pequeña
Diseño e implementación de una red LAN básica para una oficina pequeña.
- Direccionamiento IPv4 estático
- Configuración de gateway
- Gestión de switches
- Pruebas de conectividad entre dispositivos

### Lab 02 – Red Corporativa Multisede
Diseño e implementación de una red corporativa multisede (HQ + DC).
- Segmentación de redes LAN por áreas
- Enlace WAN punto a punto
- Subneteo IPv4 con VLSM
- Direccionamiento IPv6
- Enrutamiento estático
- Servidor TFTP para respaldo de configuraciones
- Pruebas de conectividad IPv4 e IPv6

## Validación
Las configuraciones fueron validadas mediante pruebas de conectividad (ping, traceroute) y el modo simulación de Packet Tracer.
